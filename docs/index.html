What It Catches

16 rules across 5 categories — every one maps to a real App Store rejection reason.

Privacy

error Missing Privacy Manifest fixable

No PrivacyInfo.xcprivacy found. Required for apps using UserDefaults, file timestamps, disk space APIs.

privacy/missing-privacy-manifest
error Missing Purpose Strings fixable

Source uses permission APIs (Camera, Location, etc.) but Info.plist is missing the matching NS*UsageDescription key.

privacy/missing-purpose-strings
warning Overbroad Permissions fixable

Permission keys declared in Info.plist with no matching API usage in your source — Apple flags these.

privacy/overbroad-permissions
error Missing ATT Consent fixable

Ad/tracking SDK detected but no ATTrackingManager consent dialog found. Required since iOS 14.5.

privacy/missing-att-consent

Metadata

error Missing App Icons

No AppIcon.appiconset with valid Contents.json and image assets found.

metadata/missing-icons
warning Version Mismatch fixable

MARKETING_VERSION in Xcode project doesn't match package.json version.

metadata/version-mismatch
error Missing Launch Screen

No LaunchScreen.storyboard found in the ios/ directory. Apple requires a launch screen.

metadata/missing-launch-screen

Payments

error Missing Restore Purchases fixable

IAP library detected but no restorePurchases call found. Apple requires a restore mechanism.

payments/missing-restore-purchases
warning External Payment Links fixable

References to external payment URLs (Stripe, PayPal, Patreon, etc.) that may trigger rejection for directing users outside the App Store.

payments/external-payment-links

Security

error Insecure Transport (ATS) fixable

NSAllowsArbitraryLoads is enabled — disables App Transport Security entirely.

security/insecure-transport
warning Hardcoded Secrets fixable

API keys, tokens, or secrets hardcoded in source files. These get bundled into the app binary and can be extracted.

security/hardcoded-secrets

Functionality

error Missing Account Deletion fixable

Auth/signup flows detected but no account deletion mechanism found. Required since June 2022.

functionality/missing-account-deletion
warning Missing Deep Link Handling fixable

URL schemes registered in Info.plist but no Linking handler in source. Unhandled deep links cause crashes.

functionality/missing-deep-link-handling
warning Minimum OS Version fixable

Deployment target is below iOS 15.0. Older targets increase compatibility issues and review scrutiny.

functionality/minimum-os-version
error Missing Sign in with Apple fixable

Third-party social login (Google, Facebook, etc.) detected but no Sign in with Apple. Required by Apple guidelines.

functionality/missing-sign-in-with-apple
warning WebView-Only App

App appears to be a thin WebView wrapper with minimal native UI. Apple rejects apps that are just websites bundled as apps (Guideline 4.2).

functionality/webview-only

Quick Start

1

Scan

Run from your React Native project root. No install required.

npx shipcheck
2

Review

See what's wrong and what's fixable. Preview AI prompts with --dry-run.

npx shipcheck fix --dry-run
3

Fix

Let your AI coding CLI patch the issues. Supports Claude Code and OpenAI Codex.

npx shipcheck fix --yes

Captain's Log

Configure via .shipcheckrc.json in your project root.

Configuration

.shipcheckrc.json
{ "agent": "claude", "rules": { "functionality/minimum-os-version": { "enabled": false }, "privacy/overbroad-permissions": { "severity": "error" } }, "exclude": ["**/vendor/**"] }
iosDir — override ios/ directory path
rules — disable rules or override severity
agent — preferred AI: claude, codex, or auto
include / exclude — source file globs

CI Integration

.github/workflows/shipcheck.yml
name: App Store Check on: [push, pull_request] jobs: shipcheck: runs-on: macos-latest steps: - uses: actions/checkout@v4 - uses: actions/setup-node@v4 with: node-version: '20' - run: npm ci - run: npx shipcheck --format github

The --format github flag outputs native GitHub Actions annotations that appear inline on pull requests.
